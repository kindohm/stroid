## feature suggestions

Feature ideas, from “nice quick win” to “bigger magic”:

Gameplay

Powerups: shield, rapid fire, boost, magnet pickup, asteroid freeze.
Boss asteroid waves: named giant asteroids with phases and shared health.
Combo scoring: streak multiplier for fast asteroid clears.
Friendly revive: living players can rescue eliminated players by reaching their ghost marker.
Weapon variants: spread shot, rail shot, mines, charged shot.
Environmental hazards: gravity wells, solar flares, drifting debris fields.
Lobby / Multiplayer

Ready check before host starts.
Spectator mode for direct links into active games.
Rematch vote after game over.
Room settings: asteroid density, lives, friendly collision, map size, chaos mode.
Invite copy button with toast instead of just selectable URL.
Host transfer when host leaves.
Progression

Persistent local pilot stats: games played, best score, favorite asteroid victim.
Unlockable ship colors/trails based on achievements.
Daily named asteroid pack.
Post-game medals: “most reckless,” “last pilot standing,” “tiny asteroid menace.”
UI Polish

Mini-map toggle and zoom levels.
Mobile/touch controls.
Audio: thrust hum, laser pops, crunchy asteroid split sounds.
Better game-over recap with timeline: first hit, biggest streak, final asteroid.
In-game pause/help overlay.
Best Next Bets

Ready check: small, lobby-friendly, improves flow.
Rematch vote: makes sessions sticky.
Room settings: unlocks lots of replay without huge engine work.
Powerups: biggest fun-per-code once core loop stays stable.
